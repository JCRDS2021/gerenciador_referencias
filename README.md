# gerenciador_referencias
Minimundo (Bando de Dados) - Gerenciador de Referências Bibliográficas.

DESCRIÇÃO DO SCRIPT

SEÇÃO 1: CRIAÇÃO DAS TABELAS
Na seção 1, são criadas todas as tabelas do minimundo: Perfil, Tema, Assunto, Obra. 

SEÇÃO 2: COMANDO INSERT PARA POVOAÇÃO DAS TABELAS
Na segunda seção, é utilizado o comando INSERT parapovoamento das tabelas.

SEÇÃO 3: REALIZANDO CONSULTAS
Na seção 3, são realizadas duas consultas: 

SELECT nome, tipo_perfil, email FROM perfil; <- Esta consulta seleciona os dados "nome", "tipo_perfil" e "email" da tabela "Perfil".

SELECT * FROM perfil WHERE tipo_perfil = "estudante"; <- Esta consulta seleciona todos os registros da tabela "Perfil", onde o "tipo_perfil" é igual a estudante. 

SEÇÃO 4: UTILIZAÇÃO DOS COMANDOS UPDATE E DELETE
Nesta seção, é utilizado o comando UPDATE para atualização de alguns registros. Já o comando DELETE, é utilizado para a exclusão de alguns registros, mediante uma condição específica.
